# Coinbara

Coinbara is a cozy Windows desktop coin flipper built around an AI-generated capybara coin. It turns a tiny yes-or-no moment into a warm, polished little ritual: click Flip, watch the coin move, and let Coinbara choose Heads or Tails.

The app is designed as a lightweight local Windows utility with no account, no network service, and no data collection. Its interface uses soft colors, capybara-themed coin artwork, a gentle flip animation, counters, and recent flip history to make simple decisions feel a little more delightful.
